## Usage

```bash
$ git clone https://github.com/lgsfarias/projeto14-e-book-store-back
$ cd projeto14-e-book-store-back
$ npm install
$ npm run dev
```

Create a `.env` file with the following content:

```
MONGO_URI='mongodb://localhost:27017'
MONGO_DB='mydatabase'
```

Connect to MongoDB.
